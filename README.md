This project presents a hybrid AI-based framework for predicting traffic congestion and recommending optimal routes in urban environments. The system combines Long Short-Term Memory (LSTM) networks for learning temporal traffic patterns, Graph Attention Networks (GAT) for modeling spatial relationships between connected road segments, and a Reinforcement Learning agent (PPO) for selecting efficient routes based on predicted conditions. It also integrates YOLOv8-based vehicle detection from CCTV feeds to estimate real-time traffic density and improve prediction accuracy. The system is implemented using Python, PyTorch, Flask, and a web interface built with HTML, CSS, and JavaScript. Designed using traffic data from Electronic City, Bengaluru, the project demonstrates how deep learning and reinforcement learning can be applied to build intelligent transportation systems for smart city applications.
